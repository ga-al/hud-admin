# HUD Admin — кастомизация admin-шаблона (portfolio)

Статичное демо кастомизации коммерческого admin-шаблона [HUD](https://www.seantheme.com/hud/) под crypto-dashboard RuExmo: tweetdeck-колонки, фильтры, карточки новостей, адаптив. Без бэкенда — контент и данные демонстрационные.

**Live demo:** https://ga-al.github.io/hud-admin/

## Стек

- HTML5
- Bootstrap 5 (через HUD Admin template)
- SCSS → CSS (исходная сборка шаблона; в репозитории — готовый `dist`)
- jQuery, ApexCharts (из шаблона)

## Что кастомизировано

- Главная страница переработана под мультиколоночный dashboard (стиль TweetDeck)
- Колонки: Новости ENG/RUS, Лидеры, Трейдеры, Биржи, Боты, Проекты
- Русскоязычный контент, фильтры и accordion-панели в колонках
- Sidebar упрощён — убраны демо-страницы шаблона

## Примечание

Базовый шаблон — коммерческий HUD Admin (SeanTheme). В репозитории только **кастомизированная страница** и минимально необходимые ассеты для демо. Исходники сборки (`src/`, `node_modules/`, gulp) и ~40 демо-страниц шаблона удалены.

Контент в колонках — демо-заглушки на момент вёрстки (2021–2022).
